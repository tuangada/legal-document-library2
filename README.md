# legal-document-library2
Website ini bertujuan untuk menjadi arsip digital terpusat (digital legal repository) yang dapat digunakan untuk menyimpan, menelusuri, dan memantau dokumen hukum secara sistematis. Dengan sistem berbasis web, seluruh data tersimpan secara real-time dan dapat diakses melalui perangkat apapun tanpa memerlukan instalasi tambahan.
